    colonel_kurtz_ex v0.1.0 API Reference

    Modules

    ColonelKurtz.BlockType

    The BlockType module defines a macro that is used to mix in the Block Type behavior. Block Types are embedded Ecto Schemas that conform to the Validatable protocol.

    ColonelKurtz.BlockTypeView
    ColonelKurtz.BlockTypes

    Provides methods for marshalling data into named BlockType structs.

    ColonelKurtz.EctoBlocks

    Implements a custom Ecto.Type (https://hexdocs.pm/ecto/Ecto.Type.html#content) that models a serializable list of blocks stored as JSON.

    ColonelKurtz.EctoHelpers

    Ecto helpers such as format_error for formatting errors.

    ColonelKurtz.FormHelpers

    Form helpers for phoenix.

    ColonelKurtz.Renderer

    Provides a utility for rendering blocks. Requires the application to configure :block_views for ColonelKurtz.

    ColonelKurtz.Validation

    ColonolKurtz.Validation provides validate_blocks/2 and validate_blocks/3 which are used to validate block data.
